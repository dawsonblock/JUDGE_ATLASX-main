# MEMORY BOUNDARIES

## Role of Memory
Memory stores are derivative indexes/caches for retrieval and operator productivity.

## Non-Authority Rule
- Memory is not authoritative evidence.
- Memory entries must not supersede or overwrite canonical evidence facts.
- Any memory-derived output must be re-grounded to source snapshots before publication.

## Allowed Uses
- Retrieval acceleration.
- Operator triage/context assembly.
- Non-binding drafting assistance.

## Disallowed Uses
- Autonomous publication decisions.
- Legal conclusions without evidence citation.
