# docs/history/

This directory holds documents that accurately described a past state of the
project but are no longer the authoritative source of truth.

Files here are kept for audit purposes.  Do not delete them — they form part
of the change-history record.  Do not cite them in new code or documentation
without noting the date and the fact that they have been superseded.

| File | Originally described | Superseded by |
|---|---|---|
| `IMPLEMENTATION_COMPLETE.md` | Declared Phase 4-6 roadmap items complete as of the initial integration spike | `CURRENT_STATUS.md`, `REPO_REALITY.md`, per-phase test suites |
| `PHASES_4_5_6_ROADMAP.md` | Roadmap for phases 4, 5, and 6 before those phases were executed | `docs/RELEASE_BLOCKERS.md` phase tracking |
